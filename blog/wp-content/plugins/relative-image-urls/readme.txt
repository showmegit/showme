== Description ==

The Relative Image URLs Plugin does the following.  When inserting an image into Wordpress posts, Wordpress uses absolute URLs to the files.  This plugin overrides that functionality and forces it to use relative URLs.

For example:

Wordpress Default:
http://www.example.com/wp-content/upload/04/20/example.jpg

Plugin Override:
/wp-content/upload/04/20/example.jpg

This is particularly useful if you plan on switching domains ever and has been requested in several help threads.

== Installation ==

   1. Install the plugin either via FTP or Admin backend
   2. Activate the plugin
